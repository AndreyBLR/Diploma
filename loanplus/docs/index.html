Main File Structure

All the file are well organized, its so easy to work with the template.
1. Unzip the files.
2. Open "loanPlus-html" folder.
3. You will find your all html pages then choose "assets" folder for all the main files.
4. Also you will see 4 directories, css, js, fonts, img.
5. In the css folder content you will find style.css and other styling files.
6. In the js folder content you will find jquery.js and other js files.
7. Fonts folder contains essetial font files for fonts.
